The "Attribute Manager" plugin includes this plugin + ability to assign attributes to multiple products at once

NOTE: You must install the main "WooCommerce - excelling eCommerce" plugin in order to use this plugin

Insert a text list of multiple attributes at once

== Installation ==

1. Unzip/Extract the plugin zip to your computer
2. Upload `\woocommerce-bulk-insert-attributes\` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to WP Admin -> Products -> Attributes -> Add an attribute (such as "Color") -> Configure Terms
5. Paste your list of options into the Bulk Insert field; with each option on a new line
